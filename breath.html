腹式呼吸以及暖嗓



介紹

就像是吹奏笛子一樣
身為一個歌唱者，我們的樂器就是自己的喉嚨以及聲帶，而空氣的來源就是呼吸
氣流通過聲帶產生共鳴，然後操控口腔嘴型來發出不一樣的音色。


Note

核心概念 : 產生氣流提供足夠的動力，來驅動聲帶產生共鳴，唱出歌聲。
歌聲 = 聲帶*呼吸*共鳴。



人體樂器的正確使用方式

曾經有沒有覺得跟別人去唱完KTV，到最後總是很累，音色也跑掉了，喉嚨非常緊呢?
說話跟唱歌都會消耗肌耐力，就算用正確的方式發聲，長期使用還是需要適當休息
(tip:睡飽能讓橫膈膜附近肌群休息，暫且不要說話讓聲帶修復，補充水分)。

Note

因為不可能無窮的唱歌以及講話下去，因此正確的方式就是 " 找尋最有效率的方式，製造最亮麗的音色 "。




最有效率的呼吸 : 腹式呼吸



為目前最有效率的歌唱呼吸方式。



如何感受腹式呼吸?


其實人在躺平休息時，最自然的呼吸就是腹式呼吸。
呼吸的時候肩膀不要聳起，胸口不要有起伏，可以透過撫摸劍突來感受橫膈膜肌群的用力。

    訓練方式
  • 透過彈唇練習習慣持續吐氣的感受
  • 吸氣八分飽，慢慢吐氣發出"s"的聲音持續一分鐘，最後用盡力氣保持發出"s"聲音把氣都吐光
  • 吸氣八分飽。慢慢透過嘴唇吐氣，力道為手掌能感受到氣流即可，除了腹部之外其他肌群不能代償，維持60秒吐完，慢慢推展到150秒吐完。 進階版:嘴巴張開，透過聲帶閉合當氣流閥門。

暖嗓

就如運動前要暖身一樣，在唱歌前要先暖嗓，讓身體習慣發聲動作，找回肌肉記憶，才能唱得更高更宏亮，更重要的是保養你的歌唱年齡，尤其是對於時常唱高強度高音以及吼腔的人。

暖嗓的方式百百種，其中最為人廣知的就是彈唇練習:透過適當的嘴型，讓你在安全的情況下習慣在歌唱時用腹式呼吸來提供氣流，也可以檢驗你是否有提供足夠的空氣。


Note

所以暖嗓不只保護你的喉嚨，建立肌肉記憶，也可以檢驗你是否用正確的方式發聲。
